# CRUD
Project to perform crud operation

Initial setup
1 - npm init --y
2 - tsc --init
3 - npm install @types/node
4 - npm install -g ts-node
5 - nodemon


To run
 npm start

To setup swagger
 npm install swagger-ui-express swagger-jsdoc

Run swagger
  http://localhost:4011/docs

Setup testing
 npm i chai chai-http mocha supertest

Run testing
 npx mocha test.js
